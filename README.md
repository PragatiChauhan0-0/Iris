# Iris-
Students often miss important academic emails. A streamlined solution is needed to ensure timely, clear, and actionable academic notifications.

## Key Features:
- Converts long or complex emails into concise, easy-to-read summaries powered by Google Gemini 2.0 flash
- Automatic reading of selected academic emails according to user preference
- Compiles extracted key points, deadlines, and action items into a message
- Sends summaries directly to a messaging app (Telegram)
- Reduces overload for students by allowing automatic filtering of important academic emails
- Minimizes misunderstanding of instructions 

## Tech Stack
- Python
- Dependencies:
-- google-generativeai
-- python-telegram-bot
-- python-dotenv

## Setup

1. Clone the repository
2. Create a virtual environment
3. Install dependencies
4. Configure environment variables
5. Run the application
